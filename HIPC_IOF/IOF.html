Specific Aims

  • SA1. Identify and prioritize data analysis use cases, and provide analytical solutions

    Comprehensive gating of all cell subsets defined within specific marker panels (T, B, Treg, etc.)

  • SA2. Critical assessment of data processing algorithms using FITMaN lyoplate data through the FlowCAP project

  • SA3. Provide a user-friendly interface implementing the data analysis pipelines (Year 2)

Overview of current tools

  • R/Bioconductor

    • flowWorkspace 2.0 for parsing workspaces (flowJo Mac and Windows)
    • flowViz for visualizing flow cytometry data (including gates) conditioning on metadata
    • Package for defining workflows (specific for each marker panel)
    • Available from Bioconductor and/or GitHub

  • Mixed effect models for accessing automated results (against manual consensus)

  • Preliminary work on web interface using LabKey

Towards a web interface

  • LabKey

      Used by CHAVI, HVTN, CAVD, ITN
    • R and JavaScript APIs
    • Existing flow cytometry module
    • Open source!

Benchmarking and standardizing analysis tools for high dimensional single cell assays

Project leader: Raphael Gottardo
  • Members: Greg Finak, Renan Sauteraud, Andrew McDavid
Co-Investigators:
  • HIPC: Steve de Rosa (FHCRC), Nick Haining (Emory), Holden Maecker (Stanford)

Specific Aims

  • SA1. Identify and prioritize assay platforms and use cases

    Flow cytometry, CyTOF, Fluidgm

  • SA2. Survey and quantitatively compare methods for single cell data analysis

  • SA3. Develop new analytic and visualization approaches to characterize and quantify heterogeneity in complex mixtures of cells

  • SA4. Implement the best analytical solutions into R packages

Why single cell?

  • The immune system is composed of many small and functionally distinct cell subsets that can only be measured accurately using single cell assays

  • The characterization of these small cell subsets requires sophisticated bioinformatics tools

  • Higher dimensional assays

    • Flow cytometry can get up to 8-10 functional markers
    • CyTOF 50+ markers
    • Fluidigm 96 genes

    With 50 markers we have many putative cell subsets


    $$2^{50}=1125899906842624$$

Short term goals

  • Work with HIPC investigators to define use cases and gather datasets

  • Leverage of existing MIMOSA (Mixture MOdel for Single Assays) framework

    Bayesian hierarchical Dirichlet-Multinomial mixture model that uses prior information to cope with the high dimensionality

  • Leverage our flow cytometry visualization framework

    • Single cell assay data model
    • Stores data and metadata in a single object
    • Use reshape and ggplot2 for plotting
